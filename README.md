# sales-data-analysis
Sales data analysis project using Python (Pandas, Matplotlib) with insights on trends, category performance, and profit relationships.
# Sales Data Analysis Project
## Objective
Analyze sales trends and derive business insights using Python.
## Tools Used
- Python
- Pandas
- Matplotlib
## Key Insights
- Highest sales observed on Friday
- Electronics category generated maximum revenue
- Positive correlation between sales and profit
